版本: v2.2 (2026-06-10)
性质: HC 数值体系顶层抽象总结（独立于 SGN）
核心原则: 本文档提炼三个数值子空间的几何本质、运算能力、工程定位与关键约束。HC 体系是一个通用数值规范，SGN 仅使用其中的 仅 level 和 合体（时间变量）子集。所有详细定义以核心规范为准。

一、几何与拓扑性质

维度 仅 HC 宇宙 仅 level 宇宙 合体宇宙
几何模型 有限深度 256 叉树（超度量） 一维整数线（阿基米德） 无限深度逃逸树（非阿基米德）
拓扑性质 有限、紧致（深度 = 1 + L） 无限、非紧 无限、非紧
映射连续性 局部常值映射 线性映射（保序连续） 不连续（进位撕裂）
索引结构 Trie 索引（前缀树） 哈希/数组索引 混合索引：level 分区 + HC Trie
聚类形态 硬聚类（超度量球分明） 软边界/梯度 层次跳跃（跨 level 不保序）
典型用途（HC 体系） 模板匹配、像素、速度、置信度 计数、调度、步数、轮次 时间戳、累计量、大跨度存档
SGN 是否使用 ❌ 不使用（仅占位） ✅ 空间变量核心 ✅ 时间变量核心（预留）

二、运算能力谱系

运算 仅 HC 宇宙（含投影沙盒） 仅 level 宇宙 合体宇宙
加法/减法 带进位（饱和/回绕），调用 sgn_hc8_add_saturated 普通整数加减 完整进位链（可向 level 溢出），调用 sgn_leveled_add
乘法 卷积+压缩（饱和），调用 sgn_hc8_mul 普通整数乘法 ⚠️ 展平‑卷积‑压缩（理论可行，工程禁止使用）
除法 投影‑欧氏‑逆归一化（PC 端扩展，依赖浮点沙盒） 普通整数除法（截断） ❌ 未定义 / 不推荐
梯度更新 float64/float128 沙盒（仅 PC 端扩展，供整数神经网络社区） 整数步进（level += 1） ❌ 不推荐（非保序，撕裂结构）
负空间 独立 sign 字段（沙盒）或基 256 补码（硬件） 自然有符号（int32_t） 有符号 level（少见）
曲线交点 投影‑求根（需连续函数，PC 端扩展） 无 无
开方 2‑adic / Hensel 提升（PC 端验证器） 无 无
幂运算 快速幂（O(log n) 乘法） 无 无

注释 1：仅 HC 宇宙的“除法/梯度/开方/幂”等连续域运算仅限 PC 端（通过投影沙盒借用 float64/float128 或整数 Hensel），且不属于 SGN 核心必需，是为整数神经网络社区提供的扩展能力。SGN 核心不使用这些功能。
注释 2：合体模式严禁用于线性映射、除法、梯度等运算；其设计目标仅是大范围高精度存储与带进位追加（如时间戳、累计量）。

五、关键约束速查

约束 描述 引用规范

六、一句话总结

· 仅 HC 宇宙：离散树结构 + 超度量 + 投影沙盒（PC 扩展）→ 保结构的同时获得连续运算能力（整数神经网络社区使用，SGN 不用）。
· 仅 level 宇宙：一维整数线 + 阿基米德序 → 简单、保序、无穷范围（SGN 空间变量核心）。
· 合体宇宙：无限逃逸树 + 进位撕裂 → 大范围存档专用，运算雷区（SGN 时间变量核心，预留）。

三个宇宙各司其职，共同构成 HC 数值体系从 MCU 到 PC、从模板匹配到超大数值的统一基础。SGN 仅选用其中的 仅 level 和 合体（时间变量）子集。
