## 1. 形式化模型

### 1.1 HC+level 表示法

一个 HC+level 数记为有序元组：

```

X = (level, int_part, frac[0], frac[1], ..., frac[L-1])

```

其中：
- `level`: 非负整数，理论无界，负责**向外**扩展整数范围（每级跨越 256）
- `int_part`: 0–255（uint8_t），负责承载单级内的整数壳
- `frac[i]`: 0–255（每层 uint8_t），负责**向内**展开小数精度
- `L`: 编译期常量 `SGN_HPDC_FRAC_LAYERS`（2/4/6）

**物理值公式**（唯一权威定义）：

```

V = level × 256 + int_part + Σ_{i=0}^{L-1} frac[i] / 256^{i+1}

```

**值域**：
- 仅 HC 模式（`level=0`，固定）：`[0, 256)`，永不达到 256（设计如此，非缺陷）
- 合体模式：`[0, ∞)`，无上限

### 1.2 字典序比较（唯一权威定义）

定义 `X < Y` 当且仅当存在索引 `k` 使得：
- 对所有 `i < k`，`x_i = y_i`
- `x_k < y_k`

比较顺序固定为：`level` → `int_part` → `frac[0]` → `frac[1]` → ... → `frac[L-1]`

**特性**：前缀式短路，首层不等即终止，平均只需比较 1~2 层。

## 2. 超度量距离（唯一权威定义与证明）

### 2.1 定义

对于 `X ≠ Y`，定义首次差异索引：

```

k(X,Y) = min{ i | x_i ≠ y_i }

```

则距离为：

```

d(X,Y) = 256^{-k(X,Y)}

```

约定 `d(X,X) = 0`。`α = 256` 为分量基数。

### 2.2 强三角不等式（唯一权威证明）

**定理**：上述距离 `d` 满足非负性、对称性、同一性，以及**强三角不等式**：

```

∀X,Y,Z:  d(X,Z) ≤ max(d(X,Y), d(Y,Z))

```

**前提**：本证明假设所有元组具有相同的固定深度 L（即仅 HC 模式，或合体模式下 level 相同且 int_part+frac 层数固定）。对于合体模式中 level 不同的数，超度量距离仍按字典序定义，且强三角不等式依然成立（因为首次差异层可能是 level 层）。合体模式的主要问题是进位导致的“无限深度树”，此时超度量球可能撕裂，详见 §5.3。

**证明**：

1. 非负性、对称性、同一性由定义直接得。

2. 设 `X,Y,Z` 为任意三个元组。令：
```

p = k(X,Y),   q = k(Y,Z),   r = k(X,Z)

```

3. **关键引理**：`r ≥ min(p, q)`。

   反证：假设 `r < min(p, q)`。由 `r < p` 知 `X` 与 `Y` 前 `r` 层相等；由 `r < q` 知 `Y` 与 `Z` 前 `r` 层相等。因此 `X,Y,Z` 三者前 `r` 层全部相等。但 `r = k(X,Z)` 意味着 `x_r ≠ z_r`，而 `x_r = y_r`（因 `p>r`）且 `y_r = z_r`（因 `q>r`），矛盾。故引理得证。

4. 于是：
```

d(X,Z) = 256^{-r} ≤ 256^{-min(p,q)} = max(256^{-p}, 256^{-q}) = max(d(X,Y), d(Y,Z))

```
   得证。

5. 因此 `d` 是一个**超度量**（ultrametric）。

### 2.3 推论：所有三角形均为等腰

由强三角不等式，对任意三点 `X,Y,Z`，`d(X,Y), d(Y,Z), d(X,Z)` 中最大的两个必然相等。

## 3. 数值验证（唯一权威案例集）

以下案例使用 `α=256`，仅考虑 `level=0` 以简化。

### 案例 1：同 int_part，不同 frac[0]

```

A = (0, 100, 50, 0, ...)
B = (0, 100, 60, 0, ...)
C = (0, 101, 0, 0, ...)

```

- `k(A,B) = 2` → `d(A,B) = 256^{-2} ≈ 1.5259e-5`
- `k(A,C) = 1` → `d(A,C) = 256^{-1} ≈ 0.00390625`
- `k(B,C) = 1` → `d(B,C) = 0.00390625`

验证：`0.00390625 ≤ max(1.5259e-5, 0.00390625) = 0.00390625` ✓

### 案例 2：深层差异

```

A = (0, 50, 10, 20, 0, ...)
B = (0, 50, 10, 30, 0, ...)
C = (0, 50, 20, 0, 0, ...)

```

- `k(A,B) = 3` → `d = 256^{-3} ≈ 5.96e-8`
- `k(A,C) = 2` → `d ≈ 1.53e-5`
- `k(B,C) = 2` → `d ≈ 1.53e-5`

验证：`1.53e-5 ≤ max(5.96e-8, 1.53e-5) = 1.53e-5` ✓

### 案例 3：不同 level

```

A = (1, 0, 0, ...)
B = (2, 0, 0, ...)
C = (3, 0, 0, ...)

```

- `k(A,B) = 0` → `d = 1`
- `k(B,C) = 0` → `d = 1`
- `k(A,C) = 0` → `d = 1`

验证：`1 ≤ max(1,1) = 1` ✓

## 4. 与算术运算的独立性说明

上述证明**仅依赖于字典序比较规则**，与加法、乘法、进位等算术运算无关。因此即使未来 HC+level 的算术实现发生变化（例如进位规则调整、引入除法），**超度量比较距离始终成立**，只要比较仍遵循字典序。这是该体系的稳定数学内核。

## 5. 分离模式几何（唯一权威解释）

### 5.1 三种独立的数学宇宙

| 模式 | 结构 | 运算规则 | 数学性质 | 几何模型 |
|------|------|---------|---------|---------|
| **仅 level** | `uint32_t` | 普通整数加法、乘法、比较 | 保序、连续、阿基米德 | 一维整数线 |
| **仅 HC** | `(int_part, frac[])` | 带进位加法，int_part 溢出截断/饱和 | 超度量比较、局部线性映射可控 | **有限深度 256 叉树** |
| **合体** | `level × 256 + HC` | 完整带进位，int_part 溢出向 level 进位 | 非保序、线性映射不连续 | **无限深度树（逃逸结构）** |

### 5.2 仅 HC 模式：有限深度树（安全舱壁）

当 `level` 被固定为 0 且不参与运算时，数值空间退化为**有限深度**的 256 叉树：
- 深度 = `1 (int_part) + SGN_HPDC_FRAC_LAYERS`
- 超度量距离 `d(X,Y) = 256^{-k}`，其中 `k ∈ {1, 2, ..., L+1}`
- 由于树深度有限，空间在超度量下是**完备且紧的**

**线性映射的局部行为**：乘以常数 `c` 的进位链最多向上传播 `⌈log_{256} c⌉` 层，不会跳出树的根节点（因为没有 level 层作为逃逸出口）。因此映射在每一层内是**局部常值**的——小球内的点经过映射后仍落在同一父分支下。

### 5.3 合体模式：无限深度树（逃逸结构）

合体模式下，`level` 为树增加了**无限向上的逃逸通道**：
- 当 HC 的 `int_part` 溢出时，进位不是终止，而是"长出"一个新的 level 节点
- 一个低位的小扰动可以通过进位链无限向上传播，最终逃逸到根节点的兄弟分支，**撕裂超度量球**

**数学性质总结**：
- 合体模式下的超度量距离定义（按字典序）本身是良定义的，但**当 int_part 溢出进位到 level 时，原本在同一超度量球内的点可能被分裂到不同 level 分支**，导致超度量球撕裂。
- 因此，合体模式不保证“有限深度树”的紧致性，也不保证加法运算下的球结构稳定性。仅 HC 模式（level 固定为 0）才是真正意义上的超度量紧空间。

**几何比喻**：
- 仅 HC：一棵被截断的圣诞树，装饰球只能在树内滚动
- 合体：一棵无限生长的世界树，底部的一个蚂蚁咬断丝线，可能导致顶端的星辰坠落

### 5.4 工程选择的拓扑依据

| 需求 | 推荐模式 | 拓扑理由 |
|------|---------|---------|
| 需要硬聚类、Trie 索引、超度量 NN | 仅 HC | 有限深度树，球面稳定 |
| 需要大范围计数、单调调度 | 仅 level | 整数线，保序连续 |
| 需要同时满足大范围+高精度 | 合体 | 无限树，但需容忍非保序性 |
| 需要线性变换（缩放、衰减） | 仅 HC 或仅 level | 避免进位逃逸到无限层 |
