## 0. 问题起源

当前 SGN 是**离散事件系统**（步进、锁定、合并），无连续时间动力学：

```python
# sgn_core.py
def train(self, signature, label):
    # 单步原子操作：匹配 -> 竞争 -> 更新 -> 衰减
```

但以下远期场景需要**时序记忆模型**：

| 场景 | 为什么需要连续时间 | 当前规避 |
|------|-----------------|---------|
| **时序预测** | 根据过去 10 步的模式预测下一步 | 无时间序列能力 |
| **记忆巩固** | 高频概念随时间自发强化，低频自发消退 | 被动衰减（>>=1） |
| **概念漂移** | 环境变化时概念缓慢迁移（如 "0" 的写法演变） | 无迁移机制 |
| **注意力波动** | 神经元响应速度随时间随机涨落 | 固定 base |
| **睡眠重放** | 离线时随机重放记忆片段以巩固 | 无离线模式 |

**核心矛盾**：若引入浮点随机微分方程（SDE）或 Langevin 动力学，SGN 的 MCU 无 FPU 能力和确定性将崩塌。

超度量布朗运动恰好解决此矛盾：在 256 叉树上定义纯整数、确定性的伪随机游走，用 LCG 生成转移概率，全程无浮点。

## 1. 形式化模型

### 1.1 超度量树上的随机游走

设 HC 树深度为 `L`，节点 `u` 在第 `d` 层（`d=0` 为根）。

**转移规则**：
```
P(u -> child_i) = 1/256   对每个存在的子节点 i
P(u -> parent)  = 1 - n_u/256   （n_u 为子节点数）
P(u -> sibling) = 0   （超度量树中不允许横向跳跃）
```

**关键性质**：
- **深层漂移**：在叶子附近（`d` 大），子节点少（`n_u` 小），回退概率高 -> 粒子在细节层"振动"
- **浅层锚定**：在根附近（`d` 小），子节点多（`n_u` 大），向下概率高 -> 粒子被"吸入"概念分支
- **概念稳定性**：一旦粒子进入某概念子树（如数字 '0'），横向逃逸到兄弟子树（如 '1'）的概率为 0，只能通过回退到父节点再重新选择

### 1.2 首达时与生成函数

设 `G_{u,v}(z)` 为从节点 `u` 到 `v` 的首达时生成函数。

**递推关系**：
```
G_{u,v}(z) = (z/256) * Σ_{w ∈ children(u)} G_{w,v}(z)   (u ≠ v)
G_{v,v}(z) = 1
```

**定理**：从根 `r` 到深度 `d` 节点 `v` 的首达时分布为：
```
P(T_{r,v} = d) = (1/256)^d
```

*证明*：每步必须选择正确的子节点（概率 1/256），连续 `d` 步独立。∎

### 1.3 与生物记忆的类比

| 生物现象 | 超度量布朗运动对应 | SGN 实现 |
|---------|------------------|---------|
| **膜电位波动** | 深层字节随机扰动 | `frac[5]` 每步 LCG 置换 |
| **LTD/LTP** | 停留时间长的路径强化 | 命中计数器 + Δ |
| **概念巩固** | 浅层节点高停留概率 | 核心模板标记 |
| **睡眠重放** | 离线时从根开始随机游走 | 夜间模式：无输入，纯游走 |
| **遗忘曲线** | 深层扰动累积导致细节丢失 | 软阈值衰减 |

## 2. 算法实现（接口声明）

### 2.1 单步转移

```cpp
// sgn_brownian.h

// 粒子状态
typedef struct {
    uint8_t  path[6];   // 当前路径（从根到当前节点）
    uint8_t  depth;     // 当前深度 0~6
    uint16_t seed;      // LCG 伪随机种子（确定性）
} brownian_particle_t;

// LCG 伪随机
uint8_t lcg_step(uint16_t* seed);

// 单步转移：返回新深度和新路径
// 实现要点：
//   1. LCG 生成伪随机数 rand ∈ [0,255]
//   2. 调用 sgn_trie_find_child() 定位当前节点（sgn_trie_core §2）
//   3. 获取子节点数 n_child = node->child_count
//   4. 若 rand < n_child：走向子节点（调用 sgn_trie_find_child() 按索引获取）
//   5. 否则：回退到父节点（depth--）
// 禁止重复实现：所有 Trie 节点查找调用 sgn_trie_core 标准接口
void sgn_brownian_step(const HCTrieNode* template_trie,
                       brownian_particle_t* p);
```

> **实现引用**: `sgn_brownian_step()` 内部的所有 Trie 节点查找调用 [`sgn_trie_core.md`](sandbox:///mnt/agents/output/sgn_trie_core.md) §2 的 `sgn_trie_find_child()`。禁止在本文档中重复实现 Trie 遍历代码。

### 2.2 睡眠重放模式（离线巩固）

```cpp
// 夜间模式：无外部输入，粒子在模板库中随机游走，强化停留路径
void sgn_sleep_replay(sgn_core_t* core, uint16_t steps) {
    brownian_particle_t p;
    p.depth = 0;
    p.seed = core->step_count;  // 确定性种子

    for (uint16_t i = 0; i < steps; ++i) {
        sgn_brownian_step(&core->template_trie, &p);

        // 强化当前停留的模板
        if (p.depth > 0) {
            // 调用 sgn_trie_find_path() 将路径转为模板 ID（sgn_trie_core §2.2）
            uint16_t tid = sgn_trie_find_tid_by_path(&core->template_trie,
                                                      p.path, p.depth);
            if (tid != 0xFFFF) {
                sgn_lru_hit(&core->templates[tid].hit_counter);
            }
        }
    }
}
```

> **实现引用**: 路径到模板 ID 的转换调用 [`sgn_trie_core.md`](sandbox:///mnt/agents/output/sgn_trie_core.md) §2.2 的 `sgn_trie_find_path()`。禁止重复实现路径遍历。

## 3. 性能与动力学特征

### 3.1 停留时间分布

| 深度 | 子节点数典型值 | 回退概率 | 平均停留步数 | 含义 |
|------|-------------|---------|------------|------|
| 0（根） | 10 | 246/256 = 96% | 1.04 | 瞬间被吸入概念 |
| 1（概念层） | 5 | 251/256 = 98% | 1.02 | 概念内稳定 |
| 3（子概念） | 3 | 253/256 = 99% | 1.01 | 子类稳定 |
| 5（细节层） | 1 | 255/256 = 99.6% | 1.004 | 细节振动 |
| 6（叶子） | 0 | 256/256 = 100% | 1 | 立即回退 |

**结论**：粒子几乎不会停留在叶子（立即回退），而是在概念层和细节层之间做"上下振动"，符合生物记忆的"巩固-遗忘"循环。

### 3.2 概念切换频率

| 事件 | 理论概率 | 1000 步中预期次数 | 生物类比 |
|------|---------|-----------------|---------|
| 根->概念A | 1/10 | ~100 | 注意力转移 |
| 概念A->概念B | 需回退再选择：~1/256 | ~4 | 思维跳跃 |
| 概念A内子类切换 | 1/5 | ~200 | 细节调整 |
| 细节层随机扰动 | 1/256 | ~1000 | 电位噪声 |

## 4. 误区澄清（v1.1 修订）

| 误区 | 事实 |
|------|------|
| "布朗运动是随机的，破坏 SGN 确定性" | **不破坏**。LCG 伪随机是**确定性**的——同种子产生同轨迹，md5sum 可复现。 |
| "和球面反射重复：都是扰动" | 不重复。球面反射是**值扰动**（改变 HC 字节），布朗运动是**路径扰动**（在树中游走）。 |
| "叶子节点 100% 回退，永远无法停留" | 正确。但"停留"通过**命中计数器强化**实现，而非粒子位置。粒子是探针，模板是地标。 |
| "需要真随机数生成器（TRNG）" | 不需要。LCG 对模拟已足够，且确定性是 SGN 的核心价值。 |
| "超度量布朗运动没有扩散方程" | 是。超度量空间上的"扩散"是**分层跳跃**，不满足标准热方程。但首达时生成函数 `G(z)` 提供了完整的概率描述。 |
| "SGN 现在就需要这个吗" | **不是**。当前 SGN 是静态快照。这是**远期时序扩展**的数学框架，为睡眠重放、概念漂移、注意力模型保留接口。 |

## 6. 总结

> **超度量布朗运动不是让 SGN 变随机，而是让记忆有温度。**

- 256 叉树上的分层随机游走：深层振动（细节遗忘），浅层锚定（概念巩固）
- 确定性 LCG：同种子同轨迹，md5sum 可复现，不破坏 SGN 核心哲学
- 首达时生成函数 `G(z)`：完整的概率动力学描述，支持统计推断
- 生物类比丰富：膜电位波动、LTD/LTP、睡眠重放、概念漂移全部映射
- 与 LRU 计数器、软阈值衰减、核心标记形成完整记忆生命周期
- 纯整数、无浮点、无 TRNG，8051 可离线执行睡眠重放
- **去重完成**：所有 Trie 遍历调用 `sgn_trie_core` 标准接口，禁止重复实现

一句话记住：

**静态快照是标本，布朗运动是活物；标本不会忘，活物才会记。**
